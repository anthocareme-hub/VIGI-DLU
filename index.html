Copier le code de VIGIE DLU

Étape 1 : copie le code de l'app avec le bouton ci-dessous.

Étape 2 : colle-le sur GitHub

  1. Va sur github.com dans Safari, connecte-toi.
  2. Ouvre ton dépôt vigie-dlu.
  3. Appuie sur ton fichier index.html existant pour l'ouvrir.
  4. Appuie sur l'icône crayon (✏️) en haut à droite pour l'éditer.
  5. Appuie longuement dans la zone de texte → "Tout sélectionner" → "Couper" (pour vider l'ancien contenu).
  6. Appuie de nouveau dans la zone, maintenant vide → "Coller" (le nouveau code que tu as copié à l'étape 1).
  7. Descends en bas de la page → "Commit changes".

C'est tout !

Ton lien GitHub Pages reste exactement le même qu'avant — donc si tes données étaient sur cette adresse, elles restent intactes. Attends 1 minute, puis recharge ton icône sur l'écran d'accueil.
